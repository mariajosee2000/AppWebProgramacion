## 🚀 Cómo Empezar

### Prerrequisitos
- Node.js 16+
- Ionic CLI: `npm install -g @ionic/cli`
- Git

### Instalación
```bash
# Clonar el repositorio
git clone https://github.com/IsaiasMr2000/tp-grupo-9.git

# Entrar al directorio
cd pedidos-fast-app

# Instalar dependencias
npm install

# Ejecutar en navegador
ionic serve
